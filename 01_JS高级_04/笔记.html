第一章 JS高级 第四天 一、闭包(函数) 1.什么是闭包? 就是一个函数 父函数里的内部函数(子函数),使用了父函数的局部变量,并且被返回到父函数外 2.作用域 全局:易污染,一直被保存在内存里 局部:不易被污染,用完即销毁 解决一个问题: 变量:不易被污染,又可以一直留在内存里 闭包:解决局部变量一直留在内存里 3.怎么使用闭包? 1)父函数内有'局部变量'(形参) 2)子函数使用了'局部变量' 3)返回子函数: return 子函数 4.使用场景 注意:不能滥用闭包,否则会导致内容泄露,内存溢出; 1)原生dom操作,获取当前dom下标 2)使用私有变量 二、Error错误(类型) 程序异常/错误 常见错误类型 1) SyntaxError：语法错误 2) ReferenceError：引用错误 3) TypeError：类型错误 4) RangeError：范围错误 - js编译遇到错误提示，后面的代码都不执行 三、异常处理 针对于 异常的 处理 使用 try catch 语法: 什么时候用? 自己写的程序:建议还是if else 遇到一些无法预知的,使用了其他的框架,可以try测试运行,catch捕获异常,可以看到错误问题,去解决 四、编程思维 1.常见的编程模式/编程思想 面向过程:注重细节,一步一步来执行 面向对象:注重功能,分功能实现 2.区分: 以前书写的dom操作的都属于面向过程:非常繁琐,后期维护非常困难 jquery框架就是采用面向对象的思维方式 来编写的:方便,不需要关心过程 3.总结 面向过程:不需要思考,直接一步执行,效率低 面向对象:前期思考需求,设计完,直接调用,效率高 五、对象 object 1.什么是对象? 描述一个具体事物 有属性(特性)和方法(功能) 每个对象都有自己的属性和方法,互不影响 2.如何创建对象? 3.访问属性或方法
